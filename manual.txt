## SECTION 1 - CUSTOMER-FACING SYSTEM OVERVIEW

The AMK Smart Pump & Compressor Control System is a dual-mode controller for:
- Water pump automation
- Air compressor control with solenoid unloader valve
- Voltage protection
- Dry-run protection
- Cloud monitoring and remote control
- Local web dashboard
- Physical manual button control
- Myanmar and English interface support

Core customer promise:
- The device protects the motor from dry-run and unsafe voltage.
- The pump still works locally even if internet, cloud, or license service is unavailable.
- Cloud and AI services are convenience features; they do not make the hardware useless when expired.

Hardware summary:
- ESP32-S3 controller
- LCD 20x4 I2C display
- ZMPT101B voltage sensor
- Waterproof ultrasonic sensor for tank level
- Non-contact flow sensor
- Relay output for pump or compressor motor
- Solenoid output for compressor unloader valve
- RGB status LED
- Physical push button

## SECTION 6 - SENSOR AND PROTECTION LOGIC

### Voltage Protection

The device filters voltage readings to avoid reacting to one-time spikes. It uses validation, median filtering, outlier rejection, smoothing, and resume delay.

Customer explanation:
- A brief power spike should not immediately chatter the relay.
- Persistent unsafe voltage stops the motor.
- The motor resumes only after voltage is stable again.

### Tank Level Sensor

The ultrasonic sensor estimates water level from measured distance.

If the LCD shows sensor error:
- Check sensor wiring.
- Check sensor power.
- Check that the sensor is not blocked or covered by foam/debris.
- Press the button to silence the alarm if needed.
- Contact support if the error continues.

### Flow / Dry-Run Detection

The device does not immediately alarm at motor start. It allows a priming period and tolerates intermittent flow.

Dry-run sequence:
1. Motor runs.
2. No flow is detected for longer than the configured delay.
3. Device enters dry-run alarm.
4. Alarm continues for a short period.
5. If not reset, the device locks for protection.
6. User can reset by physical button or dashboard.
7. If auto-retry is enabled, the device can retry after the selected time.

AI response rule:
- For dry-run questions, ask the user to check water source, pipe priming, flow sensor wiring, and whether valves are open.

## SECTION 15 - COMPRESSOR MODE

When Operating Mode is Air Compressor:
- The solenoid valve works as an unloader valve.
- Before motor start, solenoid opens for the configured delay.
- Motor starts after pre-venting.
- After motor stop, solenoid opens again for post-venting.
- Manual toggle is blocked while venting.

Customer explanation:
- VENTING is normal in compressor mode.
- Wait a few seconds until venting completes.
- Do not bypass the solenoid sequence.
